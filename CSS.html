기본 문법

CSS는 선택자와 선언으로 이루어져 있다. 선택자는 스타일을 적용할 HTML 요소를 선택한다. 선언은 선택한 요소에 스타일을 적용한다. 선언은 속성과 값으로 이루어져있다. 선택자는 ,를 사용해서 여러 개를 선택할 수 있다.

기본 문법은 아래와 같다. 여기서 p는 선택자이고 {}는 선언이며, 그 안에 있는 color는 속성이며 red는 값이다.

p {
  color : red ;
}

선택자 순서

선택자는 스타일을 적용할 HTML 요소를 선택한다. 선택자는 우선순위가 있으며, 우선순위는 선택자의 순서에 따라 결정된다.

선택자 순서는 아래 순서대로 적용된다. 결과적으로 인라인 스타일 속성이 가장 마지막에 적용된다.

  1. 태그 선택자
  2. 클래스 선택자
  3. 아이디 선택자
  4. 인라인 style 속성

box model

HTML 태그는 block 레벨의 태그와 inline 레벨의 태그로 나누어진다.

block 레벨의 태그는 한 줄을 모두 차지하는 태그이다. inline 레벨의 태그는 한 줄에 여러 개를 배치할 수 있는 태그이다. display 속성을 이용해서 태그의 레벨을 변경할 수 있다.

grid

grid는 2차원 형태의 레이아웃을 만들 수 있는 속성이다.

grid는 display: grid; 속성을 사용해서 만들 수 있다.

반응형 디자인

반응형 디자인은 화면 크기에 따라 레이아웃을 자동으로 조절하는 디자인이다.

media query(@media)를 사용해서 반응형 디자인을 구현할 수 있다.
