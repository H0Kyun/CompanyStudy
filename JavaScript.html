이벤트

이벤트는 사용자가 웹페이지와 상호작용할 때 발생하는 모든 것을 의미한다. 이벤트는 사용자가 버튼을 클릭하거나, 마우스를 움직이거나, 키보드를 누르는 등의 동작을 할 때 발생한다. 주로 `on*`형태의 이벤트가 존재한다.

태그 선택

태그를 선택하는 방법에는 여러 가지가 있다: - document.querySelector('태그'): CSS 선택자로 첫 번째 요소 선택 - document.querySelectorAll('태그'): CSS 선택자로 모든 요소 선택 - document.getElementById('아이디'): ID로 요소 선택 - document.getElementsByClassName('클래스'): 클래스로 요소들 선택 - document.getElementsByTagName('태그명'): 태그 이름으로 요소들 선택
